<h2>DESCRIPTION <a href="#toc" name="description">&uarr;</a></h2>

The tool graphicaly renders Oyjl JSON UI files using QML.

&nbsp;&nbsp;oyjl -X json+command | oyjl-args-qml -i - -c +
